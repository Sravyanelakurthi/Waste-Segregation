# Smart Waste Segregation System

An automated waste segregation system using Arduino Uno, Ultrasonic Sensor, and Soil Moisture Sensor to classify waste as Wet or Dry.

## Overview
This project detects waste using an ultrasonic sensor and classifies it based on moisture level using a soil moisture sensor. The waste is automatically directed to the appropriate compartment using a servo motor.

## Features
- Automatic waste detection
- Wet/Dry classification
- Servo-based segregation
- LCD real-time display
- Cost-effective implementation

## Components Used
- Arduino Uno
- Ultrasonic Sensor
- Soil Moisture Sensor
- Servo Motor
- LCD Display
- Jumper Wires
- USB Cable

## Working Principle
1. Ultrasonic sensor detects waste.
2. Moisture sensor checks moisture level.
3. Arduino processes data.
4. Servo motor directs waste to correct bin.
5. LCD displays classification.
